# Enhanced Anime1

![](/assets/screenshot.png)

一款适用于 [anime1.me](https://anime1.me/) 网站的增强型浏览器插件，支持 chrome、firefox。

## 特点

- 主页面为每个条目展示观看进度
- 打开番剧列表时，自动定位至上次看到的话数
- 播放番剧会自动跳转至上次观看的进度
- 圆形浮动部件，可以一览所有看过的番剧

## 下载

- Chrome: 插件地址（审核中）
- Firefox: [插件地址](https://addons.mozilla.org/addon/enhanced-anime1/)
